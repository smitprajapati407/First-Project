 🏥 First_Project – Hospital Management System (Django)

A web-based Hospital Management System built using Django.
This project helps manage patients, doctors, and appointments efficiently through a simple and user-friendly interface.

---

 🚀 Features

* 👨‍⚕️ Doctor Management (Add / View / Delete)
* 🧑‍🤝‍🧑 Patient Management (Add / View / Delete)
* 📅 Appointment Booking System
* 🔄 CRUD Operations for all modules
* 📊 Organized data handling using Django ORM
* 📱 Responsive UI using Bootstrap

---

 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite
* **Architecture:** Django MVT Pattern

---

 ⚙️ Setup Instructions

### 1. Clone the repository

git clone https://github.com/smitprajapati407/First_Project.git
cd First_Project
cd first

---

### 2. Create virtual environment

python -m venv venv
venv\Scripts\activate

---



### 3. Run migrations

python manage.py migrate

---

### 4. Run the development server

python manage.py runserver

 📌 Project Modules

* **Patients:** Add, view, and manage patient records
* **Doctors:** Manage doctor details and specialization
* **Appointments:** Schedule and manage appointments between patients and doctors
